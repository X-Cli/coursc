  \framesubtitle{\subsecname~: else if~II}

  L'exemple précédent peut être compacté en~:
  \begin{exampleblock}{Compactage grâce au block else if}
    \begin{verbatim}
if(condition1) {
  //instructions 1
}
else if (condition 2) {
  //instructions 2
}
else {
  //instructions 3
}\end{verbatim}
  \end{exampleblock}
  \par
  Il est possible de chainer autant de block \verb|else if| que nécessaire.
\end{frame}
